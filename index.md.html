# placekitten.jcbhmr.com Use this service to get random kitten images of specified dimensions. URL template: ``` https://placekitten.jcbhmr.com/<width>/<height> ``` Example URLs: ``` https://placekitten.jcbhmr.com/100/100 https://placekitten.jcbhmr.com/200/200 https://placekitten.jcbhmr.com/600/400 https://placekitten.jcbhmr.com/628/1254 ``` These URLs will return a random kitten image that fits the specified dimensions well.
